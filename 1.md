```mermaid
flowchart TD
    Inicio((Inicio)) --> IntroducirDatos[El usuario introduce sus datos en la Vista]
    IntroducirDatos --> EnviarFormulario[Pulsa el botón de 'Registrar']
    EnviarFormulario --> ValidarFormatos[La capa de Negocio valida el formato de los datos]
    
    ValidarFormatos --> Validacion1{¿Campos válidos?}
    Validacion1 -- "No (ej: email malformado)" --> MostrarError[Mostrar mensaje de error de formato]
    MostrarError --> IntroducirDatos
    
    Validacion1 -- "Sí" --> ComprobarDuplicado[El SA pide al DAO buscar el email/usuario]
    ComprobarDuplicado --> ConsultaBD[(Consulta a Base de Datos)]
    ConsultaBD --> Validacion2{¿El usuario existe?}
    
    Validacion2 -- "Sí" --> MostrarError2[Mostrar error: 'El usuario ya está registrado']
    MostrarError2 --> IntroducirDatos
    
    Validacion2 -- "No" --> GuardarUsuario[El DAO inserta el nuevo Jugador]
    GuardarUsuario --> InsercionBD[(Insertar en Base de Datos)]
    InsercionBD --> Confirmacion[La Vista muestra mensaje de registro exitoso]
    Confirmacion --> Redirigir[Redirigir a la pantalla de Login]
    Redirigir --> Fin(((Fin)))

    %% Estilos
    classDef startend fill:#333,stroke:#333,stroke-width:2px,color:#fff;
    classDef bd fill:#f9f,stroke:#333,stroke-width:2px;
    classDef error fill:#ffcccc,stroke:#ff0000,stroke-width:2px;
    
    class Inicio,Fin startend;
    class ConsultaBD,InsercionBD bd;
    class MostrarError,MostrarError2 error;
